# useImagePreload

图片预加载专用 Hook，极简版本 - 自动触发加载，无返回值。适用于需要提前加载图片资源的场景，提升用户体验。

## 安装

```bash
npm install @lerna-demo/shared
```

## 基本用法

```vue
<template>
  <div>
    <img :src="imageUrl" alt="预加载图片" />
  </div>
</template>

<script setup>
import { useImagePreload } from '@lerna-demo/shared'

const imageUrl = 'https://example.com/large-image.jpg'

// 基本用法 - 立即预加载
useImagePreload(imageUrl)
</script>
```

## 延迟加载

```vue
<script setup>
import { useImagePreload } from '@lerna-demo/shared'

const imageUrl = 'https://example.com/hero-image.jpg'

// 延迟2秒后开始预加载
useImagePreload(imageUrl, {
  delay: 2000
})
</script>
```

## 带回调的用法

```vue
<template>
  <div>
    <div v-if="loading" class="loading">图片加载中...</div>
    <div v-if="error" class="error">{{ error }}</div>
    <img v-if="loaded" :src="imageUrl" alt="图片" />
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { useImagePreload } from '@lerna-demo/shared'

const imageUrl = 'https://example.com/image.jpg'
const loading = ref(true)
const loaded = ref(false)
const error = ref('')

useImagePreload(imageUrl, {
  onLoad: () => {
    loading.value = false
    loaded.value = true
    console.log('图片预加载成功')
  },
  onError: (err) => {
    loading.value = false
    error.value = err.message
    console.error('图片预加载失败:', err)
  }
})
</script>
```

## 批量预加载

```vue
<script setup>
import { useImagePreload } from '@lerna-demo/shared'

const imageUrls = [
  'https://example.com/image1.jpg',
  'https://example.com/image2.jpg',
  'https://example.com/image3.jpg'
]

let loadedCount = 0
const totalImages = imageUrls.length

imageUrls.forEach((url, index) => {
  useImagePreload(url, {
    delay: index * 500, // 错开加载时间
    onLoad: () => {
      loadedCount++
      console.log(`已加载 ${loadedCount}/${totalImages} 张图片`)
    },
    onError: (err) => {
      console.error(`图片 ${url} 加载失败:`, err)
    }
  })
})
</script>
```

## API 参考

### useImagePreload(url, options?)

**参数:**

- `url`: `string` - 图片URL地址
- `options`: `ImagePreloadOptions` - 配置选项

**Options:**

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|---------|
| priority | `'high' \| 'low'` | - | 加载优先级（预留字段） |
| delay | `number` | `0` | 延迟加载时间（毫秒） |
| onLoad | `() => void` | - | 加载成功回调 |
| onError | `(error: Error) => void` | - | 加载失败回调 |

**返回值:**

无返回值（void）

## 使用场景

### 1. 首屏图片预加载

在页面加载时预加载首屏关键图片，提升用户体验：

```vue
<script setup>
import { useImagePreload } from '@lerna-demo/shared'

// 预加载首屏轮播图
const bannerImages = [
  '/images/banner1.jpg',
  '/images/banner2.jpg',
  '/images/banner3.jpg'
]

bannerImages.forEach(url => {
  useImagePreload(url)
})
</script>
```

### 2. 懒加载优化

结合懒加载，提前预加载即将进入视口的图片：

```vue
<script setup>
import { ref, onMounted } from 'vue'
import { useImagePreload } from '@lerna-demo/shared'

const nextImages = ref([])

onMounted(() => {
  // 预加载下一屏的图片
  nextImages.value.forEach(url => {
    useImagePreload(url, {
      delay: 1000 // 延迟1秒预加载
    })
  })
})
</script>
```

### 3. 图片切换优化

在图片切换前预加载下一张图片：

```vue
<template>
  <div>
    <img :src="currentImage" alt="当前图片" />
    <button @click="nextImage">下一张</button>
  </div>
</template>

<script setup>
import { ref, watch } from 'vue'
import { useImagePreload } from '@lerna-demo/shared'

const images = [
  '/images/1.jpg',
  '/images/2.jpg',
  '/images/3.jpg'
]

const currentIndex = ref(0)
const currentImage = computed(() => images[currentIndex.value])

// 预加载下一张图片
watch(currentIndex, (newIndex) => {
  const nextIndex = (newIndex + 1) % images.length
  useImagePreload(images[nextIndex])
}, { immediate: true })

const nextImage = () => {
  currentIndex.value = (currentIndex.value + 1) % images.length
}
</script>
```

## 最佳实践

1. **合理使用延迟加载**: 避免同时加载大量图片影响页面性能
2. **错误处理**: 始终提供 onError 回调处理加载失败的情况
3. **批量加载控制**: 批量预加载时建议错开加载时间
4. **内存管理**: 预加载的图片会缓存在浏览器中，注意内存使用
5. **网络优化**: 在网络较差的环境下考虑降低预加载的图片数量

## 注意事项

- 该 Hook 为极简版本，自动触发加载且无返回值
- 图片加载成功后会自动缓存在浏览器中
- 跨域图片需要服务器支持 CORS
- 建议在组件挂载后使用，避免不必要的资源消耗
